# Grok Insider

Open-source tools for developers and power users — fast, native, self-hostable.
Mostly Rust and Python, built and shipped on NixOS.

## Projects

| Project | What it is | Stack |
|---------|-----------|-------|
| [**open-media**](https://github.com/grok-insider/open-media) | Watch movies, series & anime from the terminal via Real-Debrid / P2P → mpv | Rust |
| [**open-subtitle**](https://github.com/grok-insider/open-subtitle) | One subtitle engine — find, score, download, sync, translate & transcribe (CLI, daemon, C-ABI, mpv) | Rust |
| [**open-recorder**](https://github.com/grok-insider/open-recorder) | Medal.tv / ShadowPlay-style NVENC game clipper for Linux: instant replay, overlay HUD, clip editor | Rust |
| [**open-usage**](https://github.com/grok-insider/open-usage) | AI coding-subscription usage tracker (CLI + daemon + Waybar) across 17 providers | Rust |
| [**open-sessionbar**](https://github.com/grok-insider/open-sessionbar) | Desktop-agnostic status-bar monitor for OpenCode sessions (HTTP/SSE + live TUI) | Rust |
| [**open-scrapper**](https://github.com/grok-insider/open-scrapper) | Modern OSS web-scraping & botting framework, AI-assisted with local models | Python |
| [**open-proxy**](https://github.com/grok-insider/open-proxy) | Gather, analyze & export free public proxies, with a SQLite learning loop | Python |
| [**release-changelog-action**](https://github.com/grok-insider/release-changelog-action) | Shared GitHub Action: git history → AI-written release notes via OpenRouter | GitHub Action |

Every Rust/Python project ships prebuilt artifacts on GitHub Releases through one
automated, AI-changelog release pipeline.
